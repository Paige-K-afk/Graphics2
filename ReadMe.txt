DirectXFramework is the files that can be used to re-make this project.
Release is the release build and contains the executable.

How to run:
Go into Release file. Run Graphics2.exe

Controls:
W - forward
A - turn left
S - backward
D - turn right
L Shift - Look Up
CTRL - Look Down
E - Hold to show wireframe (please look at the wireframe.
			    the texture on the terrain is transparent and looks flat
			    but the wireframe shows all the details)
